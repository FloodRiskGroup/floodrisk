Benvenuti alla documentazione di FloodRisk¶

FloodRisk¶

Cos’è FloodRisk ?

FloodRisk è un plugin per QGIS che consete la stime delle conseguenze di un’inondazione, in termini di perdite di vite e di danni economici diretti.

Il termine RISCHIO : Rischio = (Probabilità) x (Conseguenze)

FloodRisk riguarda la stima delle conseguenze

Il Plug-in¶

Contenuti:

  • Gestione Progetto
  • Gestione Database
    • Esempio di tabella di Tipi di Bene
    • Esempio di tabella di ListaTipiCurveAltezzaDanno
    • Esempio di tabella di curva Altezza-Danno
    • Esempio di tabella di FatalityRates
    • Esempio di tabella FloodSeverity
  • Danni Economici Diretti
  • Stima delle perdite di vite
